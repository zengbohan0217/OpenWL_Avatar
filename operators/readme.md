This package contains operator pipelines for OpenWL-Avatar.
- gen_game_cg: pipeline for generating game CG (cinematic) content
- gen_ue_avatar: pipeline for generating and importing avatars into Unreal Engine 5
- process_input: pipeline for processing and preprocessing user inputs (text, image, video)

从 models 中调用模型进行功能实现，如果一个功能太复杂，尽量多弄几个文件，让代码好看懂一些，方便后续开发
